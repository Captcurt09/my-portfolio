## ğŸš€ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Captcurt09/my-portfolio.git
   ```

2. Install dependencies:
   ```bash
   cd my-portfolio
   npm install
   ```


